Pirates of the Caribbean Adventure

Project Addendum

Process and Creative Choices:

For this project, I chose a "Pirates of the Caribbean" theme because it offers rich storytelling potential and a lot of exciting decisions for the player to make. I wanted to build an interactive game with suspenseful moments, such as fighting sea monsters, finding cursed treasure, and making alliances with pirates.

The structure of the game is based on decision-making at key points in the story. Each decision leads to different paths, which either result in an exciting adventure or a grim fate. I made sure to include at least 8 different endings, as required, so that the game feels varied and replayable.

I used JavaScript objects to store the different stages of the story. Each stage contains text, choices, consequences, and an associated image, which makes it easy to update the story dynamically. The CSS gives the page a "piratey" feel, with dark colors, yellow accents, and an old-timey vibe reminiscent of a pirate ship.

Challenges Faced and Solutions:

One of the biggest challenges was ensuring that the story paths aligned correctly. There are many branching decisions, so I had to carefully track each choice and make sure the consequence led to the right next stage in the story. I solved this by using a clear structure in the JavaScript file where each stage is linked to the next one through objects.

Another challenge was ensuring the images changed dynamically based on the choices the player made. This required some careful manipulation of the DOM in JavaScript. I created a function to update both the story text and the images based on the current stage of the game.

Overall, the project was a great opportunity to practice working with JavaScript, HTML, and CSS to create an interactive and engaging web experience. I learned how important it is to plan out the flow of the game before starting to code, and I gained a better understanding of how to manage user interactions in a dynamic webpage.
