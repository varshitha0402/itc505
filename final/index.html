Lights Out Game

Instructions:
- Click on squares that are Orange to toggle their state.
- The goal is to turn all squares Light Blue.
- Choose a difficulty level to start the game.

Addendum

Steps Taken to Build the Game:

  1. Planned the structure of the game, ensuring it could dynamically generate a grid for different levels.
  2. Implemented the toggling logic, where clicking a square changes its state and the states of its adjacent squares.
  3. Ensured solvable random configurations by simulating clicks instead of assigning random states directly.
  4. Added a winning condition check to detect when all squares are toggled to the "off" state.
  5. Designed the user interface with a modern 3D aesthetic using CSS, inspired by the Lights Out 3D game.
  6. Integrated reset functionality to allow players to restart the current level without refreshing the page.
  7. Included pop-ups to notify players when they win and encourage them to start a new game or try a different level.
  8. Created a detailed Addendum to explain the steps taken and challenges encountered during development.

The Lights Out game combines logic, aesthetics, and usability, providing an engaging experience while showcasing web development skills. The combination of CSS for styling, JavaScript for logic, and HTML for structure made it a successful project.
